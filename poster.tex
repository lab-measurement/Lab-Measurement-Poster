\item

in your looong C program?
\item
Use a text processing language to manage your measurement! Use Perl!
\end{itemize}

\begin{itemize}
\item
LinuxGPIB and all hardware supported by it
\item
Linux USB-TMC kernel driver
 \item 
NI-VISA (Windows and Linux) and all hardware supported by it
 \item
Oxford Instruments IsoBus
 \item
Raw TCP connection, generic network socket
 \item
RS232 serial port
\end{itemize}

\heading{Internal architecture}

\vspace*{-3mm}
\placefigure{file=images/structure}
\begin{itemize}
 \item
 Lab::Bus --- hardware driver encapsulation
 \item
 Lab::Connection --- thin glue layer
 \item
 Lab::Instrument --- instrument-level logic, sending/receiving command strings
 \item 
 Lab::XPRESS --- sweep and frame classes for nested loop measurements
\end{itemize}
